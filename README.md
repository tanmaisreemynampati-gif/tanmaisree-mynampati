## 📌 Featured project

### 🏠 Irish property price analysis & prediction (2010–2021)
> **Python · pandas · scikit-learn · statsmodels · Prophet · GeoPandas · Folium**

An end-to-end machine learning project analysing Irish residential property sales from the official Property Price Register.

- 🧹 Full data cleaning pipeline: Unicode normalisation, feature engineering, outlier handling
- 🤖 Trained & compared 3 ML models: Random Forest, Gradient Boosting (tuned), Linear Regression
- 📈 Built 6 forecasting models projecting Irish property price trends to **2030**
- 🗺️ Interactive choropleth maps showing county-level price disparities across Ireland
- 📊 Full EDA suite: correlation heatmaps, seasonal trend analysis, quarterly transaction volumes

**Best model:** Tuned Gradient Boosting Regressor — R² = 0.29 · MAE = €18,500 · RMSE = €27,300
